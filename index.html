Daniel Zamora Páez

Como desarrollador web, poseo una sólida base en el manejo de tecnologías HTML, CSS y JavaScript, lo que me permite crear interfaces atractivas, funcionales y adaptadas a diferentes dispositivos. En el ámbito del desarrollo backend, tengo experiencia trabajando con Python utilizando el framework Flask para la creación de aplicaciones ligeras y escalables, así como con Java empleando Spring Boot para el desarrollo de soluciones robustas y seguras. Además, cuento con conocimientos en el diseño, implementación y optimización de bases de datos relacionales, especialmente con MySQL, utilizando XAMPP como entorno de desarrollo local para la gestión y prueba de aplicaciones.

Habilidades

Frontend

Desarrollo de interfaces modernas, responsivas y optimizadas para la experiencia del usuario.

Backend

Creación de aplicaciones seguras, escalables y orientadas a buenas prácticas de programación.
